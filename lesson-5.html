Циклы

35 мин Начальный

Что такое циклы?

Циклы позволяют выполнять один и тот же блок кода многократно. Это мощный инструмент для автоматизации повторяющихся задач и обработки коллекций данных.

Цикл for

Цикл for используется, когда известно точное количество итераций:

for (инициализация; условие; обновление) {
    // тело цикла
}

Пример: Вывод чисел от 1 до 5

for (int i = 1; i <= 5; i++) {
    System.out.println("Число: " + i);
}

Цикл while

Цикл while выполняет блок кода, пока условие истинно:

while (условие) {
    // тело цикла
}

Пример: Обратный отсчет

int count = 5;
while (count > 0) {
    System.out.println("Осталось: " + count);
    count--;
}
System.out.println("Старт!");

Цикл do-while

Цикл do-while похож на while, но гарантирует по крайнер мере одно выполнение тела цикла:

do {
    // тело цикла
} while (условие);

Пример: Запрос ввода до получения правильного значения

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
int number;

do {
    System.out.print("Введите положительное число: ");
    number = scanner.nextInt();
} while (number <= 0);

System.out.println("Спасибо! Вы ввели: " + number);

Сравнение циклов

Тип цикла Когда использовать Преимущества Недостатки
for Когда известно количество итераций Компактная запись, все элементы управления в одном месте Менее гибкий, чем while
while Когда количество итераций неизвестно Гибкость, условие проверяется перед выполнением Легко создать бесконечный цикл
do-while Когда нужно по крайнер мере одно выполнение Гарантирует по крайнер мере одно выполнение Условие проверяется после выполнения

Управление циклами: break и continue

Ключевое слово break используется для преждевременного выхода из цикла:

for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // выход из цикла при i = 5
    }
    System.out.println(i);
}
// Вывод: 1, 2, 3, 4

Ключевое слово continue пропускает текущую итерацию и переходит к следующей:

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // пропуск итерации при i = 3
    }
    System.out.println(i);
}
// Вывод: 1, 2, 4, 5

Вложенные циклы

Циклы можно вкладывать друг в друга для работы с многомерными структурами данных:

// Вывод таблицы умножения
for (int i = 1; i <= 10; i++) {
    for (int j = 1; j <= 10; j++) {
        System.out.print(i * j + "\t");
    }
    System.out.println(); // переход на новую строку
}

Упражнение

Напишите программу, которая:

  1. Выводит все четные числа от 1 до 20 с помощью цикла for
  2. Вычисляет сумму чисел от 1 до 100 с помощью цикла while
  3. Запрашивает у пользователя числа до тех пор, пока не будет введен 0 (используйте do-while)
  4. Выводит треугольник из звездочек с помощью вложенных циклов: 
    *
**
***
****
*****
  5. Находит все простые числа от 1 до 50

Бесконечные циклы

Бесконечный цикл выполняется бесконечно, пока не будет прерван:

// Бесконечный цикл с while
while (true) {
    // код, который выполняется бесконечно
    // для выхода используйте break
}

// Бесконечный цикл с for
for (;;) {
    // код, который выполняется бесконечно
    // для выхода используйте break
}

Пример: Простое меню с выходом

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
boolean running = true;

while (running) {
    System.out.println("1. Опция 1");
    System.out.println("2. Опция 2");
    System.out.println("3. Выход");
    System.out.print("Выберите опцию: ");
    
    int choice = scanner.nextInt();
    
    switch (choice) {
        case 1:
            System.out.println("Вы выбрали опцию 1");
            break;
        case 2:
            System.out.println("Вы выбрали опцию 2");
            break;
        case 3:
            System.out.println("Выход...");
            running = false;
            break;
        default:
            System.out.println("Неверный выбор");
    }
}

Пример программы

public class LoopsExample {
    public static void main(String[] args) {
        // Цикл for - вычисление факториала
        int number = 5;
        int factorial = 1;
        
        for (int i = 1; i <= number; i++) {
            factorial *= i;
        }
        System.out.println("Факториал " + number + " = " + factorial);
        
        // Цикл while - поиск первого числа, делящегося на 7 и 9
        int n = 1;
        while (true) {
            if (n % 7 == 0 && n % 9 == 0) {
                System.out.println("Первое число, делящееся на 7 и 9: " + n);
                break;
            }
            n++;
        }
        
        // Вложенные циклы - прямоугольник из символов
        int rows = 4;
        int cols = 6;
        
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                System.out.print("*");
            }
            System.out.println();
        }
        
        // Цикл с continue - пропуск нечетных чисел
        System.out.println("Четные числа от 1 до 10:");
        for (int i = 1; i <= 10; i++) {
            if (i % 2 != 0) {
                continue;
            }
            System.out.print(i + " ");
        }
    }
}

Ключевые моменты

  • Циклы позволяют выполнять код многократно
  • Цикл for идеален, когда известно количество итераций
  • Цикл while подходит, когда количество итераций неизвестно
  • Цикл do-while гарантирует одно выполнение тела цикла
  • break используется для выхода из цикла
  • continue пропускает текущую итерацию
  • Циклы можно вкладывать друг в друга для работы с многомерными данными
  • Бесконечные циклы полезны для программ, которые должны работать постоянно
