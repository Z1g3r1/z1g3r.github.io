Переменные и типы данных

20 мин Начальный

Что такое переменные?

Переменные в Java — это контейнеры для хранения данных. Каждая переменная имеет:

  • Тип данных — определяет, какие значения может хранить переменная
  • Имя (идентификатор) — используется для обращения к переменной
  • Значение — данные, хранящиеся в переменной

Объявление переменных

Для создания переменной нужно указать её тип и имя:

тип имяПеременной;

Можно сразу присвоить значение:

тип имяПеременной = значение;

Основные типы данных в Java

Java — язык со статической типизацией, что означает необходимость объявлять тип переменной до её использования.

Примитивные типы

Тип Размер Описание Пример
byte 1 байт Целое число от -128 до 127 byte age = 25;
short 2 байта Целое число от -32768 до 32767 short temperature = -100;
int 4 байта Целое число от -231 до 231-1 int population = 1500000;
long 8 байт Целое число от -263 до 263-1 long distance = 15000000000L;
float 4 байта Число с плавающей точкой float price = 10.99f;
double 8 байт Число с плавающей точкой двойной точности double pi = 3.1415926535;
char 2 байта Один символ Unicode char grade = 'A';
boolean 1 бит Логическое значение true/false boolean isJavaFun = true;

Ссылочные типы

К ссылочным типам относятся строки, массивы, классы и интерфейсы. Самый распространенный — String:

String greeting = "Привет, мир!";

Правила именования переменных

  • Могут содержать буквы, цифры, знак подчеркивания _ и знак доллара $
  • Не могут начинаться с цифры
  • Не могут быть ключевыми словами Java (class, public, static и т.д.)
  • Чувствительны к регистру (myVar и myvar — разные переменные)

Упражнение

Создайте программу, которая объявляет переменные разных типов:

  1. Целочисленную переменную для хранения вашего возраста
  2. Переменную с плавающей точкой для хранения вашего роста
  3. Строковую переменную с вашим именем
  4. Логическую переменную, указывающую, нравится ли вам Java

Выведите значения всех переменных на экран.

Пример программы

public class VariablesExample {
    public static void main(String[] args) {
        // Объявление и инициализация переменных
        String name = "Алексей";
        int age = 16;
        double height = 1.75;
        boolean lovesJava = true;
        
        // Вывод значений переменных
        System.out.println("Имя: " + name);
        System.out.println("Возраст: " + age + " лет");
        System.out.println("Рост: " + height + " м");
        System.out.println("Нравится Java: " + lovesJava);
    }
}

Ключевые моменты

  • Все переменные должны быть объявлены перед использованием
  • Java — строго типизированный язык, нельзя изменить тип переменной после объявления
  • Имена переменных должны быть осмысленными и описывать хранимые данные
  • Для имен переменных принято использовать camelCase (например, myVariableName)
