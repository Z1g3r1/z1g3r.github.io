Массивы

40 мин Средний

Что такое массивы?

Массив — это структура данных, которая хранит фиксированное количество элементов одного типа. Элементы массива расположены в памяти последовательно, что обеспечивает быстрый доступ к любому элементу по его индексу.

Объявление и инициализация массивов

Массивы в Java можно объявлять и инициализировать несколькими способами:

Способ 1: Объявление с последующей инициализацией

// Объявление массива
тип[] имяМассива;

// Инициализация массива
имяМассива = new тип[размер];

Пример: Создание массива целых чисел

int[] numbers;
numbers = new int[5]; // массив из 5 целых чисел

Способ 2: Объявление и инициализация в одной строке

тип[] имяМассива = new тип[размер];

Пример: Создание массива строк

String[] names = new String[3]; // массив из 3 строк

Способ 3: Инициализация с предопределенными значениями

тип[] имяМассива = {значение1, значение2, ..., значениеN};

Пример: Создание массива с начальными значениями

int[] primes = {2, 3, 5, 7, 11, 13}; // массив простых чисел
String[] weekDays = {"Пн", "Вт", "Ср", "Чт", "Пт", "Сб", "Вс"};

Доступ к элементам массива

Для доступа к элементам массива используется оператор [] с указанием индекса элемента. Индексация начинается с 0.

int[] numbers = {10, 20, 30, 40, 50};

// Получение значения элемента
int first = numbers[0]; // 10
int third = numbers[2]; // 30

// Изменение значения элемента
numbers[1] = 25; // теперь numbers[1] = 25
numbers[4] = numbers[0] + numbers[2]; // numbers[4] = 10 + 30 = 40

Важно: Попытка доступа к несуществующему элементу (вне границ массива) вызовет исключение ArrayIndexOutOfBoundsException.

Длина массива

У каждого массива есть свойство length, которое содержит количество элементов в массиве:

int[] numbers = {5, 10, 15, 20};
int size = numbers.length; // size = 4

Пример: Использование length в цикле для перебора элементов массива

int[] numbers = {5, 10, 15, 20, 25};

for (int i = 0; i < numbers.length; i++) {
    System.out.println("Элемент " + i + ": " + numbers[i]);
}

Многомерные массивы

Java поддерживает многомерные массивы (массивы массивов). Наиболее часто используются двумерные массивы.

Двумерные массивы

// Объявление и создание двумерного массива
тип[][] имяМассива = new тип[строк][столбцов];

Пример: Создание и использование двумерного массива

// Создание матрицы 3x3
int[][] matrix = new int[3][3];

// Инициализация значений
matrix[0][0] = 1;
matrix[0][1] = 2;
matrix[0][2] = 3;
matrix[1][0] = 4;
matrix[1][1] = 5;
matrix[1][2] = 6;
matrix[2][0] = 7;
matrix[2][1] = 8;
matrix[2][2] = 9;

// Или с предопределенными значениями
int[][] predefinedMatrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

Перебор многомерных массивов

// Перебор двумерного массива с помощью вложенных циклов
for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}

Полезные методы для работы с массивами

Класс Arrays из пакета java.util предоставляет множество полезных методов для работы с массивами.

Метод Описание Пример
Arrays.toString() Преобразует массив в строку для вывода Arrays.toString(arr)
Arrays.sort() Сортирует массив Arrays.sort(arr)
Arrays.binarySearch() Бинарный поиск в отсортированном массиве Arrays.binarySearch(arr, key)
Arrays.fill() Заполняет массив указанным значением Arrays.fill(arr, value)
Arrays.copyOf() Копирует массив с указанной длиной Arrays.copyOf(arr, newLength)
Arrays.equals() Сравнивает два массива Arrays.equals(arr1, arr2)

Пример: Использование методов класса Arrays

import java.util.Arrays;

int[] numbers = {5, 3, 8, 1, 4};

// Сортировка массива
Arrays.sort(numbers); // [1, 3, 4, 5, 8]

// Поиск элемента
int index = Arrays.binarySearch(numbers, 5); // index = 3

// Преобразование в строку
String arrayString = Arrays.toString(numbers); // "[1, 3, 4, 5, 8]"

// Заполнение массива
int[] filledArray = new int[5];
Arrays.fill(filledArray, 7); // [7, 7, 7, 7, 7]

// Копирование массива
int[] copiedArray = Arrays.copyOf(numbers, 3); // [1, 3, 4]

Упражнение

Напишите программу, которая:

  1. Создает массив из 10 случайных чисел от 1 до 100
  2. Находит максимальный и минимальный элементы массива
  3. Вычисляет среднее значение элементов массива
  4. Сортирует массив по возрастанию
  5. Создает двумерный массив (матрицу) 4x4 и заполняет его значениями умножения индексов (matrix[i][j] = i * j)
  6. Находит сумму элементов главной диагонали матрицы

Передача массивов в методы

Массивы передаются в методы по ссылке, что позволяет изменять исходный массив внутри метода:

public class ArrayMethods {
    // Метод для умножения всех элементов массива на 2
    public static void multiplyByTwo(int[] arr) {
        for (int i = 0; i < arr.length; i++) {
            arr[i] *= 2;
        }
    }
    
    // Метод для создания и возвращения нового массива
    public static int[] createArray(int size, int value) {
        int[] newArray = new int[size];
        Arrays.fill(newArray, value);
        return newArray;
    }
    
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5};
        multiplyByTwo(numbers); // исходный массив изменяется
        System.out.println(Arrays.toString(numbers)); // [2, 4, 6, 8, 10]
        
        int[] newArray = createArray(5, 7);
        System.out.println(Arrays.toString(newArray)); // [7, 7, 7, 7, 7]
    }
}

Пример программы

import java.util.Arrays;
import java.util.Random;

public class ArraysExample {
    public static void main(String[] args) {
        // Создание и заполнение массива случайными числами
        int[] numbers = new int[10];
        Random random = new Random();
        
        for (int i = 0; i < numbers.length; i++) {
            numbers[i] = random.nextInt(100) + 1; // числа от 1 до 100
        }
        
        System.out.println("Исходный массив: " + Arrays.toString(numbers));
        
        // Поиск максимального и минимального элементов
        int max = numbers[0];
        int min = numbers[0];
        
        for (int i = 1; i < numbers.length; i++) {
            if (numbers[i] > max) {
                max = numbers[i];
            }
            if (numbers[i] < min) {
                min = numbers[i];
            }
        }
        
        System.out.println("Максимальный элемент: " + max);
        System.out.println("Минимальный элемент: " + min);
        
        // Сортировка массива
        Arrays.sort(numbers);
        System.out.println("Отсортированный массив: " + Arrays.toString(numbers));
        
        // Создание и работа с двумерным массивом
        int[][] matrix = new int[4][4];
        
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                matrix[i][j] = i * j;
            }
        }
        
        System.out.println("Двумерный массив:");
        for (int[] row : matrix) {
            System.out.println(Arrays.toString(row));
        }
        
        // Сумма элементов главной диагонали
        int diagonalSum = 0;
        for (int i = 0; i < matrix.length; i++) {
            diagonalSum += matrix[i][i];
        }
        
        System.out.println("Сумма элементов главной диагонали: " + diagonalSum);
    }
}

Ключевые моменты

  • Массивы хранят фиксированное количество элементов одного типа
  • Индексация элементов массива начинается с 0
  • Свойство length содержит количество элементов в массиве
  • Многомерные массивы - это массивы массивов
  • Класс Arrays предоставляет полезные методы для работы с массивами
  • Массивы передаются в методы по ссылке
  • Попытка доступа к элементу вне границ массива вызывает исключение
  • Цикл for и enhanced for (for-each) удобны для перебора элементов массива
