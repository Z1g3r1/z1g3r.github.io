Условные конструкции

30 мин Начальный

Что такое условные конструкции?

Условные конструкции позволяют программе принимать решения и выполнять различные действия в зависимости от условий. Они являются фундаментальной частью любого языка программирования, включая Java.

Оператор if

Оператор if выполняет блок кода, если условие истинно:

if (условие) {
    // код, выполняемый если условие истинно
}

Пример: Проверка возраста для голосования

int age = 18;

if (age >= 18) {
    System.out.println("Вы можете голосовать!");
}

Оператор if-else

Оператор if-else позволяет выполнить один блок кода, если условие истинно, и другой - если ложно:

if (условие) {
    // код, если условие истинно
} else {
    // код, если условие ложно
}

Пример: Проверка четности числа

int number = 7;

if (number % 2 == 0) {
    System.out.println("Число четное");
} else {
    System.out.println("Число нечетное");
}

Оператор if-else-if

Для проверки нескольких условий используется конструкция if-else-if:

if (условие1) {
    // код, если условие1 истинно
} else if (условие2) {
    // код, если условие2 истинно
} else {
    // код, если все условия ложны
}

Пример: Оценка результата теста

int score = 85;

if (score >= 90) {
    System.out.println("Отлично!");
} else if (score >= 75) {
    System.out.println("Хорошо!");
} else if (score >= 60) {
    System.out.println("Удовлетворительно");
} else {
    System.out.println("Неудовлетворительно");
}

Тернарный оператор

Тернарный оператор ? : позволяет compactly записать простое условие:

переменная = (условие) ? выражениеЕслиИстина : выражениеЕслиЛожь;

Пример: Определение минимального из двух чисел

int a = 10;
int b = 5;
int min = (a < b) ? a : b;

System.out.println("Минимальное число: " + min);

Оператор switch

Оператор switch позволяет выбрать один из многих вариантов выполнения кода:

switch(выражение) {
    case значение1:
        // код для значение1
        break;
    case значение2:
        // код для значение2
        break;
    // ...
    default:
        // код, если ни одно значение не совпало
}

Пример: Определение дня недели

int day = 3;
String dayName;

switch(day) {
    case 1:
        dayName = "Понедельник";
        break;
    case 2:
        dayName = "Вторник";
        break;
    case 3:
        dayName = "Среда";
        break;
    case 4:
        dayName = "Четверг";
        break;
    case 5:
        dayName = "Пятница";
        break;
    case 6:
        dayName = "Суббота";
        break;
    case 7:
        dayName = "Воскресенье";
        break;
    default:
        dayName = "Неверный день";
}

System.out.println(dayName);

Упражнение

Напишите программу, которая:

  1. Запрашивает у пользователя число (можно использовать Scanner)
  2. Проверяет, является ли число положительным, отрицательным или нулем
  3. Проверяет, делится ли число на 2 и на 3 одновременно
  4. Используя switch, выводит название месяца по его номеру (1-12)
  5. Использует тернарный оператор для определения максимального из двух чисел

Вложенные условия

Условные операторы можно вкладывать друг в друга для создания сложных логических конструкций:

int age = 25;
boolean hasLicense = true;

if (age >= 18) {
    if (hasLicense) {
        System.out.println("Вы можете водить машину");
    } else {
        System.out.println("Вам нужно получить права");
    }
} else {
    System.out.println("Вы слишком молоды для вождения");
}

Логические операторы в условиях

Для создания сложных условий используются логические операторы:

int age = 25;
int score = 85;

// Логическое И (&&) - оба условия должны быть истинны
if (age >= 18 && score >= 70) {
    System.out.println("Доступ разрешен");
}

// Логическое ИЛИ (||) - хотя бы одно условие должно быть истинно
if (age >= 65 || score >= 90) {
    System.out.println("Специальный доступ");
}

// Логическое НЕ (!) - инвертирует условие
if (!(age < 18)) {
    System.out.println("Вы совершеннолетний");
}

Пример программы

import java.util.Scanner;

public class ConditionalExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("Введите число: ");
        int number = scanner.nextInt();
        
        // Проверка на положительное/отрицательное
        if (number > 0) {
            System.out.println("Число положительное");
        } else if (number < 0) {
            System.out.println("Число отрицательное");
        } else {
            System.out.println("Это ноль");
        }
        
        // Проверка делимости
        if (number % 2 == 0 && number % 3 == 0) {
            System.out.println("Число делится на 2 и на 3");
        } else {
            System.out.println("Число НЕ делится на 2 и на 3 одновременно");
        }
        
        // Определение времени суток
        int hour = 14;
        
        if (hour >= 5 && hour < 12) {
            System.out.println("Утро");
        } else if (hour >= 12 && hour < 18) {
            System.out.println("День");
        } else if (hour >= 18 && hour < 23) {
            System.out.println("Вечер");
        } else {
            System.out.println("Ночь");
        }
        
        scanner.close();
    }
}

Ключевые моменты

  • Условные операторы позволяют управлять потоком выполнения программы
  • Оператор if выполняет код, если условие истинно
  • Конструкция if-else позволяет обработать оба случая (истина/ложь)
  • Конструкция if-else-if полезна для проверки нескольких условий
  • Оператор switch удобен для выбора из множества вариантов
  • Тернарный оператор позволяет compactly записывать простые условия
  • Логические операторы (&&, ||, !) помогают создавать сложные условия
